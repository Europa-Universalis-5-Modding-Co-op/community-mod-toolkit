# Contributing

Contributions are welcome! This is a community toolkit and anyone can add new tools, improve existing ones, or fix bugs.

1. Add your script to `tools/`.
2. Document it in the [wiki](https://github.com/Europa-Universalis-5-Modding-Co-op/community-mod-toolkit/wiki) under **Tools**. If your tool's docs are hosted elsewhere, the wiki entry can just link there.
3. Add an entry to the Tools table in this README.
4. Add any Python dependencies to `tools/dependencies/requirements.txt`.
5. Submit a pull request.
